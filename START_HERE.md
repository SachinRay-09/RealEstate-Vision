## ‚ö° Quick Demo (2 minutes)

### Step 1: Start the App
```bash
cd realestate-vision
npm install
npm run dev
```

Open: http://localhost:5173

### Step 2: Try It
1. You'll see a satellite map background with a search bar
2. Type: **"123 Main St"**
3. Click: **"INITIATE SCAN"** (cyan button)
4. Watch:
   - 5 agents activate in left panel
   - 3D model appears in center
   - Price prediction shows on right
   - Satellite map flies to location

### Step 3: Explore
- Click **"Satellite"** button to see aerial view
- Click **"3D Model"** to return
- Click **"VR Mode"** for immersive view
- Try **"New Scan"** for another property
